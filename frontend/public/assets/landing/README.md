# Landing Assets

- `landing-poster.webp` — минимальный placeholder (1×1 px) для graceful degradation, замените на финальный постер (минимум 1920×1080, WebP) перед релизом.
- Любые дополнительные материалы (видео, LUT, HDRI) помещайте сюда же и документируйте в `specs/001-modern-home-design/quickstart.md`.

> Внимание: при обновлении ассетов сохраняйте оригинальные файлы вне репозитория и не коммитьте материалы с авторскими ограничениями.
